# invoice-data-extraction
Weights file not included
Download weights and upload in yolo folder.

Input path of invoice in the program and run.

Detects different fields such as ABN, ACCOUNT INFORMATION, INVOICE DATA, TABLES, COMPANY NAME, GST, TOTAL and extracts them as seperate images.

Runs OCR on the different images extracted and stores result in results folder.

Seperate model for extracting table data from the table image.
